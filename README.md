# blog-vue.js-springboot

一个基于Vue.js 与Springboot 开发的博客项目。 前后端分离

前端技术栈： Vue.js Vue-Router Vuex Axios mavon-editor

后端技术栈： Springboot Shiro Jpa

UI框架使用了ElementUI
